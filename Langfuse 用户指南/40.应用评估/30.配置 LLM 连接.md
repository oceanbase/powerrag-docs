# 配置 LLM 连接

在 Playground、评估等功能中使用模型前，需要配置 LLM 连接。

1. 从页面左下角选择 **设置** > **LLM 连接**。
2. 单击 **+添加 LLM 连接**。
3. 填写参数，其中：
    - **API 密钥：**前往模型厂商官方网站获取模型 API key。
    - **API Base URL：**填入`https://dashscope.aliyuncs.com/compatible-mode/v1`。

![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/configure-llm.png)

4. 单击 **创建连接**。
