# 概述

应用评估与观测模块为 PowerRAG 平台提供了评测与观测能力。该模块包括两个独立但协同的服务组件：

+ **Langfuse 服务**：负责采集和分析模型运行数据，提供可视化观测、评测以及 Prompt 管理能力。
+ **Bridge 服务**：作为中间层，负责进行评测数据的处理和 RAG 应用的调用，以及对调用结果进行评估。

您需完成上述两个组件的部署后，才能在 PowerRAG 控制台中使用 **应用观测** 与 **应用评测** 功能。

## 部署依赖与前提条件
在开始部署之前，请确保：

+ 已完成 **PowerRAG 平台** 的部署与初始化。
+ 当前环境具备基础容器运行能力（Docker ）。
+ 拥有 PowerRAG 控制台访问权限。

## 部署顺序
应用评估与观测模块的部署建议顺序如下：

1. 部署 **Langfuse 服务**。具体操作，请参见 [部署 Langfuse](/10.部署与初始化/10.部署%20Langfuse%20服务.md)。
2. 在 Langfuse 控制台中 **注册用户** 并 **创建 API 密钥。**
3. 部署 **Bridge 服务**。

完成部署后，需要修改 PowerRAG 配置，配置 Langfuse 的连接和密钥信息，然后即可在 PowerRAG 控制台中访问 **观测 **和** 评测** 入口进行使用。
