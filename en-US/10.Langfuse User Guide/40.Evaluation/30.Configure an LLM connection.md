# Configure an LLM connection

Before you use a model in features like Playground and Evaluation, you need to configure the LLM connection.

1. Click **Settings** > **LLM Connections**.
2. Click **+ Add LLM Connection**.
3. Configure the parameters, including:
    - **API Key**: Visit the model provider's official website to obtain the model API key.
    - **API Base URL**ï¼šEnter `https://dashscope.aliyuncs.com/compatible-mode/v1`.

    <img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/configure-llm-en.png" width="600">

4. Click **Create connection**.
