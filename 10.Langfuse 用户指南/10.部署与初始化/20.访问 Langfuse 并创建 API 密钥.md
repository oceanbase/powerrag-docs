# 访问 Langfuse 并创建 API 密钥

Langfuse 服务通过 API 密钥实现对外认证与访问控制。Bridge 服务需使用该密钥与 Langfuse 通信，从而启用评测与观测功能。本章节介绍如何在 Langfuse 控制台中创建 API 密钥。

## 前提条件
已完成部署 Langfuse 服务。

## 操作步骤
1. 登录 PowerRAG 控制台，在左侧导航栏单击 **观测** 访问 Langfuse 控制台。  
2. 在注册页，填写 **名称**、**邮箱**、**密码**，完成新账户注册。
3. 登录 Langfuse 控制台，在左侧导航栏单击 **组织**，然后在页面右上角单击 **+新组织**。
4. 按照页面提示，分别完成组织创建、成员邀请、项目创建。
5. 在 **4. 设置跟踪**页面，单击 **创建 API 密钥**，生成您的项目密钥。
