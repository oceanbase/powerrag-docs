# Playground

直接在提示词 Playground 中测试和迭代您的提示词。调整提示词和模型参数，查看不同模型如何响应这些输入变化。这使您能够快速迭代提示词并进行优化，以便在您的 LLM 应用中获得最佳效果，而无需在工具之间切换或使用任何代码。

![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/playground1.png)

## 核心功能
### 并排比较视图
比较多个提示词变体。您可以一次性执行所有变体，也可以专注于单个变体。每个变体都保留各自的 LLM 设置、变量、工具定义和占位符，以便您立即查看每个更改的影响。

以比较变量传入不同值的影响为例：

1. 在导航栏选择 **Playground**。
2. 在页面中编辑提示词，使用`{{variable}}`的格式插入变量。
3. 插入变量后，上方将显示变量总数。单击数字，在弹窗中输入变量的一个值，单击 **提交**，即可获得相应的输出。
4. 单击右上角![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/icon3.png)图标，右侧将出现另一个输入输出窗口。再次单击变量总数，在弹窗中输入变量的另一个值，单击 **提交**，获得相应的输出。
5. 比较这两个变量值对应的输出。

### 在 Playground 打开你的提示词
您可以在 Playground 中打开在[提示词页面](../10.快速开始.md)创建的提示词。

1. 在导航栏选择 **提示词**，进入提示词列表。
2. 单击目标提示词，进入提示词详情页面。
3. 选择右上角 **Playground** >** 刷新 Playground**，将直接在 Playground 页面打开该提示词。  
若选择 **Playground** >** 添加到现有的**，Playground 页面将在现有提示词的右侧另开一个输入输出窗口打开该提示词。

### 保存提示词到提示词管理
当您对 Playground 中编辑的提示词满意时，您可以点击保存按钮将其保存到提示词管理中。

1. 单击右上角![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/icon4.png)图标，单击 **保存为新提示词**，跳转到提示词编辑页面。
2. 输入提示词名称，配置 Config、标签等。
3. 单击 **创建提示**，完成提示词的创建。

### 工具调用和结构化输出
Langfuse Playground 支持工具调用和结构化输出模式，使您能够定义、测试和验证模型的执行，模型的执行依赖于工具调用并且使用特定的响应格式。

目前，Langfuse 仅支持在 Playground 中打开 OpenAI ChatML 格式的工具类观测。

工具调用：

+ 使用 JSON 模式定义来定义自定义工具
+ 通过模拟工具响应来实时测试依赖工具的提示词
+ 将工具定义保存到您的项目中

结构化输出：

+ 使用 JSON 模式强制执行响应格式
+ 将架构保存到您的项目中
+ 使用结构化输出从 OpenAI 生成跳转到 Playground 

请联系技术支持人员了解详细的使用方法。

### 添加提示词变量
您可以在 Playground 中添加提示词变量来模拟提示词的不同输入。

操作方法请参见“并排比较视图”章节。

### 切换模型
若已将多个模型添加到 Langfuse，您可以在 Playground 中选择您喜欢的模型。如何添加模型，请参见 [配置 LLM 连接](/40.应用评估/30.配置%20LLM%20连接.md)。

1. 在模型列表选择模型。

![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/playground2.png)

2. 单击![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/icon5.png)图标，可配置模型的各项参数。

![](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/powerrag/langfuse/playground3.png)
