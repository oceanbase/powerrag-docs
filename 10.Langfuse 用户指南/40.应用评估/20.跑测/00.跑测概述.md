# 概述

跑测允许您使用数据集系统地测试您的 LLM 应用，从而评估和比较其性能。

每次跑测都基于一个包含输入和可选的预期输出的数据集。该数据集可以是本地的，也可以托管在 Langfuse 上。对于每个输入，实验都会运行一个任务功能——当通过 SDK 跑测时，任务功能就是您的 LLM 应用；当通过控制台跑测时，任务功能就是发送给模型的提示词。

评测结果将使用各种评估方法来评估和评分。

## 如何跑测？
下表显示了基于数据托管位置和跑测位置的不同跑测配置：

| **托管位置** | **Langfuse 执行** | **本地/CI 执行** |
| --- | --- | --- |
| Langfuse 数据集 | [通过控制台进行跑测](/40.应用评估/20.跑测/30.通过控制台跑测.md) | [通过 SDK 进行跑测](/40.应用评估/20.跑测/20.通过%20SDK%20跑测.md) |
| 本地数据集 | 不支持 | [通过 SDK 进行跑测](/40.应用评估/20.跑测/20.通过%20SDK%20跑测.md) |
